# INDONESIAN_STAGE_1B_CONTENT_MATRIX.md

Last updated: 2026-05-25

## Purpose

This matrix translates `docs/EXAM_CONTENT_BLUEPRINT.md` into a concrete 60-card Stage 1B planning set for Indonesian learners preparing for JFT-Basic / SSW / Work-in-Japan practical Japanese.

The goal is validation and monetization focus, not broad app completion. The sample should feel directly useful for Indonesian learners who need Japanese for exam-adjacent tasks, workplace survival, and daily life in Japan.

## Scope and guardrails

- Create planning only for 60 Stage 1B cards.
- Use Indonesian as the primary support language and English as fallback/reference.
- Keep all final card text original in the draft document.
- Do not copy official JFT sample questions, answer choices, audio scripts, Irodori text, textbook text, recalled questions, leaked questions, paid textbook examples, or proprietary workplace manuals.
- Do not create production JSON in this issue.
- Do not modify PWA code, add audio, add payment/login/subscription, or add Google Drive behavior.

## Information-search mini card structure

Information-search mini cards must be true search tasks, not translation-only cards. Each draft should include:

- notice/table text
- learner task/question in Indonesian
- English fallback task/question
- expected answer
- short usage note

These cards should train the learner to find the needed item from a short notice, table, message, or sign-like text. The Japanese text remains original draft content and must still pass human review before production use.
